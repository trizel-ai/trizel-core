# Scientific Authorizations Directory

**Path:** `docs/authorizations/`  
**Purpose:** Stores specific scientific authorization declarations  
**Governance Authority:** TRIZEL Governance (Layer-0)

---

## Overview

This directory contains specific scientific authorization records issued under the TRIZEL Scientific Authorization Framework.

Each authorization represents a limited and controlled exception to the SYSTEM FREEZE and permits scientific activity only within the strict scope defined in the authorization document.

---

## Authorization Naming Convention

Authorization documents follow the naming pattern:

```
SA-[TOPIC]-[TYPE]-[NUMBER].md
```

Where:
- **SA** = Scientific Authorization
- **TOPIC** = Brief identifier of the subject matter
- **TYPE** = Type of authorization (e.g., THEORY, EXPERIMENT, ANALYSIS)
- **NUMBER** = Sequential number with zero-padding (001, 002, etc.)

## Authorization Requirements

All scientific authorization records in this directory must:

1. Be issued through the Scientific Authorization Framework process
2. Include clear scope definition and explicit prohibitions
3. Reference the SYSTEM FREEZE status
4. Include governance authority signature
5. Be referenced in the main Scientific Authorization Declaration

---

## Revocation

Any authorization may be revoked at any time by TRIZEL Governance (Layer-0). Revoked authorizations will be moved to a `revoked/` subdirectory with updated status.
