# TRIZEL Governance (Operational)

This repository (`trizel-core`) is the **operational governance authority** of the TRIZEL ecosystem.

It defines:
- Enforcement rules
- Repository classification
- Canonical vs non-canonical boundaries
- Website and infrastructure binding contracts

This repository **does NOT** contain:
- Scientific theories
- Data
- Analysis
- Executable code
- Epistemic or constitutional definitions

### Constitutional Reference

All authoritative definitions, epistemic law, and constitutional authority are defined exclusively in:

**TRIZEL Constitution v1.0.0**  
Repository: `trizel-epistemic-engine`

This repository is **subordinate** to the Constitution and may not override, redefine, or reinterpret constitutional definitions.

### Change Policy

This document is versioned and may change **only through explicit commits** to this repository.  
Any change must remain compliant with the Constitution and must not introduce epistemic or executable authority.
