# TRIZEL Governance (Operational)

This repository (`trizel-core`) is the operational governance authority of the TRIZEL ecosystem.

It defines:
- Enforcement rules
- Repository classification
- Canonical vs non-canonical boundaries
- Website binding contracts

This repository does not contain:
- Scientific theories
- Data
- Analysis
- Executable code

Authoritative definitions and epistemic law are defined in the TRIZEL Constitution
(`trizel-epistemic-engine`).

This document is versioned and subject to change only through explicit commits.
